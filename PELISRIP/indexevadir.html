Evadir Limite MEGA

Hola, como muchos sabrán de hace un tiempo para acá MEGA empezó a limitar sus descargas, solo permite descargar alrededor de 5 a 10 GB cada cierto tiempo, este limite se lograba evadir con ayuda de la aplicación MegaDowloader, lamentablemente MEGA encontró la forma de parchar la vulnerabilidad de la cual se aprovechaba el MegaDowloader resultando también afectado por el limite, con esto bajar de MEGA se complica un poco, pero no es imposible, el método que muchos están usando es el de cambiar su IP una vez superado el límite, para ello podemos usar herramientas VPN que modifican tu IP.

Para descargar desde el navegador puedes modificar tu IP con la extensión ZenMate VPN (recomendada por el usuario disqus Fran Fran) .

Para descargar con gestor las aplicaciones Hide My IP y Hotspot Shield, te ayudaran a modificar tu IP, creo también están disponibles como extensiones para el navegador.

Si tienes buenas conexión a Internet configura algun gestor para que te baje varios enlaces en paralelo, podrás bajar varias partes simultáneamente sin restricción.

El usuario disqus Adrian CelizM encontró en un sitio varias recomendaciones para evadir el limite de MEGA, recomiendan las aplicaciones Windscribe y Ultrasurf para modificar nuestra IP y descargar desde gestores de descarga, otra alternativa que comentan es hacer uso de la aplicación Psiphon, esta aplicación enmascara nuestra conexión utilizando un sistema de VPN junto con técnicas de ocultación SSH y HTTP Proxy permitiendo también bajar desde gestores.
